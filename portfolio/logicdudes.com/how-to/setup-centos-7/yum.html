Yum

Objective

Use yum to update the server and install ntp

1. Update CentOS

Update the software on your CentOS server by using the yum command. This will check the repositories listed in /etc/yum.rempos.d/ for any updates available for your operating system and services. Adding a -y flag will automatically choose yes for all questions.

yum update

2. Install ntp

Next we will use yum to install ntp, a package that will be later used to update the servers time.

yum install ntp -y

3. Look Around

Look for packages using the yum search command.

yum search vnc
