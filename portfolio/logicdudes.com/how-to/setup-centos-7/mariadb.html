MariaDB Server

Objective

Install MariaDB Server (mysql) on CentOS 7, set an admin password, create a database and user

1. Install

Install mariadb and mariadb-server with yum

yum install mariadb mariadb-server

2. Star & Enable

Start mariadb (mysql) server with systemd

systemctl start mariadb && systemctl enable mariadb

3. Set Password

Set the password for the root mariadb user

mysqladmin -u root password ‘agoodpassword';

4. Login

Log into the database as the root user with a password using the mysql command

mysql -u root -p

5. Create Database

Create your first SQL database called myfirstdb

create database myfirstdb;

6. Create User

Grant a new user full access to your database

grant all on myfirstdb.* to myfirstuser@localhost identified by ‘anotherpassword';

7. Exit

Exit MariaDB with exit

exit
