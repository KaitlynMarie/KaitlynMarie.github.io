Launch CentOS 7 VPS

Objective:

Setup a Linux Virtual Private Server running CentOS 7

1. Create an account

First things first, you’ll need an account. Setting up an account with a virtual server host will give you a static IP attached to a Linux instance while we’ll use to setup a CentOS virtual private server. It’s virtual because it is a machine within a machine (the host), and it’s private because only you (and the hosting company) will have access to it. We recommend DigitalOcean. They offer a wide variety of options starting for as little as $5 a month. To setup an account, go to https://digitalocean.com and create a username and password. Remember, think security from the beginning! Use a secure email account (protected by 2-step auth) and a good, unique password (something that wouldn’t be found in a password dictionary).

2. Login

This step is easy. Login with your email address and that good password you just created.


3. Click Create Droplet

The Create Droplet button is on the top right side of the screen. This will take you to the VPS creation page where you’ll set a few options then launch your own Linux server.


4. Set a Hostname

The hostname includes the computer name and your domain name. For this tutorial, I will use ldt.logicdudes.com and your hostname should be yourcomputer.yourdomain.tld (if you don’t have a domain name yet, namecheap.com is great!).

5. Select Type

Here you get to spec out your new server. For our purposes, the 512MB instance will do fine. You can always change the type and all it requires is a basic restart (the whole process takes less than 2 minutes, and that includes the time taken to find your 2-step auth device :)


6. Select Region

Generally, just select the region closest to you. Eventually, once you are setting up your own DNS servers, you’ll want an instance in two separate regions to ensure proper redundancy.


7. Select Operating System

This tutorial will be about CentOS 7, so choose CentOS 7. Generally speaking, the same process can be followed to work with Fedora, however Ubuntu and Debian will require a few alterations (mostly package names and configuration file locations).


8. Receive Email

DigitalOcean will send you an email with your new CentOS VPS IP address and root password. Remember the IP (or at least write it down); don’t worry so much about the password, you will be changing it next.


9. SSH Into Your New Server

And now this is the best part. To access your new VPS, SSH into it. The username will be root and the IP is what you wrote down in step 8.

ssh root@171.182.192.5

You’ll see something like the picture below. First your machine will ask if you trust the machine you’re attempting to connect to, and yes, yes you do. Next the server will ask for your password; type the one in from the email. Next, DigitalOcean has their CentOS instances require a password change, so type in the password again, then create a new one.


That’s it!

You’re now logged into CentOS, the best server software in the world, and it’s your command line! What do you want to do now?
